# 🌿 The Green Meridian

> *Where spatial science meets the canopy*

A hyper-personalized daily morning brief for GIS professionals preparing for the GISP exam. Built with pure HTML/CSS/JS — no framework, no build step, no backend. Just open and go.

## ✨ What's inside every edition

| Section | Content |
|---|---|
| 🎯 GISP Question of the Day | Interactive exam-level question with instant feedback & explanation |
| 📡 GIS & Spatial Technology | Cutting-edge news: LiDAR, ML canopy detection, open data platforms |
| 🌳 Urban Forestry & Canopy | Canopy equity research, new datasets (GEDI, NLCD), i-Tree updates |
| 💼 Manager Tip | Rotating: stakeholder comms, Agile, mentoring, budgeting |
| ✨ Happy News | Women in STEM wins, cool science, environmental justice |
| 🇪🇸 Palabra del Día | Nature/geography-themed Spanish word of the day |

## 🚀 Setup (2 minutes)

### Option A — GitHub Pages (public URL)
1. Fork or clone this repo
2. Go to **Settings → Pages → Source: main branch / root**
3. GitHub gives you a URL: `https://yourusername.github.io/the-green-meridian`
4. Open the URL, enter your Anthropic API key in the banner, and generate!

### Option B — Local
```bash
git clone https://github.com/yourusername/the-green-meridian.git
cd the-green-meridian
open index.html   # or just double-click it
```

## 🔑 API Key

This app calls the [Anthropic API](https://console.anthropic.com) directly from your browser. You'll need a free API key.

- Your key is stored **only in your browser's localStorage**
- It is sent **only to `api.anthropic.com`** — nowhere else
- Never commit your API key to this repo

Cost per generation: roughly **$0.003–0.005** (fraction of a cent) using Claude Sonnet.
